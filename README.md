# Raspored sjedenja — Web aplikacija

Ovo je kompletna web aplikacija za izradu rasporeda sjedenja za vjenčanja.

✅ React (Vite) frontend  
✅ Supabase Auth (login + reset lozinke)  
✅ Netlify Functions backend  
✅ Drag & drop raspored  
✅ Undo opcija  
✅ Upload logotipa  
✅ Personalizirani PDF export  
✅ Print preview  
✅ Hosting na Netlify (bez instalacije)

## Tehnologije
- Netlify (frontend hosting + backend functions)
- Supabase (baza + autentifikacija + storage)
- React (UI)
- PDFKit (server-side PDF)

## Deploy
1. Kreirati Supabase projekt  
2. Importirati SQL shemu  
3. Dodati .env varijable u Netlify  
4. Povezati GitHub repo  
5. Deploy = gotovo 🎉
